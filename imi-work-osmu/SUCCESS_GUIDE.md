### H1 중복 제거
```python
def remove_first_h1(self, html_content):
    """HTML에서 첫 번째 H1 태그 제거"""
    import re
    pattern = r'<h1[^>]*>.*?</h1>'
    modified_html = re.sub(pattern, '', html_content, count=1, flags=re.IGNORECASE | re.DOTALL)
    return modified_html.strip()
```
