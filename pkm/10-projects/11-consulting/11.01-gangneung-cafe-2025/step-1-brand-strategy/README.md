# Step 1: 브랜드 전략 수립

**기간**: 2025.10 ~ 2025.11 (약 4주)
**계약 금액**: 8,000,000원
**담당**: 이림 (브랜드 전략가)

---

## 목표

강릉 카페의 브랜드 정체성을 명확히 정의하고, 시장에서의 차별화 포인트를 확립합니다.

---

## 주요 활동

### 1. 브랜드 디스커버리
- [ ] 브랜드 질문지 회수 및 분석
- [ ] 킥오프 미팅 (1차 출장)
- [ ] 클라이언트 비전/가치관 정리

### 2. 시장 조사
- [ ] 강릉 지역 카페 시장 조사
- [ ] 경쟁사 분석 (5-10개 카페)
- [ ] 타겟 고객 인터뷰 (3-5명)

### 3. 브랜드 전략 수립
- [ ] 브랜드 포지셔닝 정의
- [ ] 브랜드 아이덴티티 (Mission, Vision, Values)
- [ ] 브랜드 스토리 개발
- [ ] 타겟 고객 페르소나 (2-3개)

### 4. 전략 문서화
- [ ] 브랜드 전략 보고서 작성
- [ ] 클라이언트 프레젠테이션

## 체크리스트

### Phase 1A: 디스커버리 (Week 1-2)
- [ ] 브랜드 질문지 발송 ✅ (2025-10-15 완료)
- [ ] 브랜드 질문지 회수
- [ ] 1차 킥오프 미팅 일정 조율
- [ ] 1차 출장 진행 (강릉)
- [ ] 미팅 노트 작성

### Phase 1B: 리서치 (Week 2-3)
- [ ] 강릉 지역 카페 리스트업 (10-15개)
- [ ] 현장 방문 조사 (3-5개 카페)
- [ ] 온라인 리뷰 분석 (네이버, 인스타그램)
- [ ] 경쟁사 SWOT 분석
- [ ] 타겟 고객 인터뷰 설계
- [ ] 타겟 고객 인터뷰 실행

### Phase 1C: 전략 개발 (Week 3-4)
- [ ] 브랜드 포지셔닝 맵 작성
- [ ] 브랜드 아이덴티티 초안
- [ ] 브랜드 스토리 초안
- [ ] 타겟 페르소나 작성 (2-3개)
- [ ] 클라이언트 피드백 수렴
- [ ] 최종 전략 문서 작성

### Phase 1D: 승인 (Week 4)
- [ ] 브랜드 전략 보고서 완성
- [ ] 클라이언트 프레젠테이션 준비
- [ ] 2차 미팅 (전략 발표)
- [ ] 피드백 반영 및 최종 승인

## 주요 질문

이 단계에서 답해야 할 핵심 질문들:

1. **이 카페는 누구를 위한 공간인가?**
2. **강릉에서 왜 이 카페를 선택해야 하는가?**
3. **이 브랜드가 전하고자 하는 가치는 무엇인가?**
4. **고객은 이 카페에서 어떤 경험을 하게 되는가?**
5. **3년 후 이 브랜드는 어떤 모습으로 기억되길 원하는가?**
