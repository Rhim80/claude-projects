# 이미커피 페어링 디저트 관리

## 폴더 구조
```
pairing/
├── README.md                    # 이 파일
├── guidelines.md                # 작성 지침 원본
├── monthly/                     # 월간 페어링 아카이브
│   ├── 2025-04.md
│   ├── 2025-05.md
│   └── ...
└── weekly/                      # 주간 페어링 아카이브
    ├── week-107.md
    ├── week-108.md
    └── ...
```

## 운영 방식
- **월간 페어링**: 매월 2종 소개 → `monthly/YYYY-MM.md`
- **주간 페어링**: 격주 1종 소개 → `weekly/week-XXX.md`
